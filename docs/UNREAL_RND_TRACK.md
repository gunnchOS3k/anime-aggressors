# Unreal R&D Track

Unreal is **not** a panic replacement for Godot. It is a controlled research track for anime combat presentation.

## Goals

- Skeletal fighter rig reference
- Animation Blueprint locomotion
- Control Rig pose tests
- Niagara aura / hit VFX
- Sequencer camera hit moments
- Gameplay Ability System–style move definition study

## Location

`game/unreal/AnimeAggressorsArena/`

## Non-goals (this track)

- GitHub Pages runtime
- CI dependency on Unreal Engine install
- Blocking Godot shipping

## Acceptance

Unreal R&D is useful only if it produces **better-looking fighter reference**, **better combat animation reference**, or **better VFX reference** than current Godot placeholders — documented in `UNREAL_TO_GODOT_LESSONS.md`.

## Setup

See `game/unreal/README.md`. Requires UE 5.x locally; not required for `npm run quality`.
