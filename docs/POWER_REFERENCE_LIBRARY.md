# Power Reference Library

Inspiration, combat grammar, move-shape study, animation pacing, and fantasy archetype research for the seven Anime Aggressors fighters — **not direct copying**. Keep move names, costumes, animations, sounds, and exact sequences original.

## How to use this list

For each fighter, pick **3–5 inspiration anchors**, not one. Record anchors as **mechanic principles** in production specs (e.g. *precision ignition*, *armored commitment*), then create original moves.

Example for Ember Vale:

- Body flame brawling = rushdown pressure
- Martial fire forms = directional footwork specials
- Precision ignition = ranged snap-burst
- Fire movement = dash extensions
- Flame armor = aura intimidation windows

Original moves: `Cinder Rush`, `Flame Fang Drive`, `Ignition Palm`, `Ash Wake`, `Solar Breaker`.
