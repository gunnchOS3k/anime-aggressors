## Gamepad

- Connect controller and press any button to register.
- With two pads connected, P1/P2 assign automatically.
- Remap via **Controls / Settings** → Remap.

---

## Combat notation

| Input | Ground | Air |
|-------|--------|-----|
| Attack | Neutral / forward / up / down tilt, dash attack | Neutral / forward / back / up / down aerial |
| Special + direction | Neutral / side / up / down special | Same |
| Shield | Hold shield | — |
| Dodge | Spot dodge; left/right + dodge = roll | Air dodge |
| Grab | Grab; direction + attack while grabbing = throw | — |

---

## Known limitations

- No touch controls yet.
- P1/P2 keyboard overlap on one keyboard — use gamepads for comfortable 2P.
- P3/P4 not available in the main game flow.

See `docs/INPUT_SYSTEM.md` for engineering details.
