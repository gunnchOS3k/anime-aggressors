# Gameplay Media Assets

## Recommended files

| File | Purpose | Suggested size |
|------|---------|----------------|
| `gameplay-banner.png` | Hero banner on README | 1200×630 or 16:9 widescreen |
| `gameplay.gif` | Short loop of Play Match or Derby launch | 10–20s, under 5 MB |

## How to capture

1. Run `npm run dev` locally or use the [GitHub Pages build](https://gunnchOS3k.github.io/anime-aggressors/).
2. Record **Play Match** (knockback launch) or **Impact Dummy Derby** (Kinetic Bat hit).
3. Export GIF with a tool like ScreenToGif, Kap, or ffmpeg.
4. Export a still frame or composed banner for `gameplay-banner.png`.
5. Commit both files to this folder.
6. Update root `README.md` top links:
   - `[Watch Gameplay GIF](docs/media/gameplay.gif)`
   - `![Anime Aggressors gameplay banner](docs/media/gameplay-banner.png)`

Do not add broken image links to the README until the files exist in this directory.
