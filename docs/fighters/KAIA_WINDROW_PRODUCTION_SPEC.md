# Kaia Windrow — Production Spec

## 1. Fantasy
Spacing wind fighter — sash and wing sleeves control air drift.

## 1a. Inspiration anchors (mechanic principles)
See [POWER_REFERENCE_LIBRARY.md](../POWER_REFERENCE_LIBRARY.md#4-kaia-windrow--wind-air-storm-pressure-flight).

- Air-column dominance and aerial drift
- Curved wind blade projectiles
- Push/pull spacing control
- Lingering cyclone multi-hits
- Anti-edgeguard float tools

## 2. Silhouette
Athletic, flowing sash, wing sleeve accents, green/teal palette.

## 3. Body type
Medium-light — high air mobility.

## 4. Movement identity
Floaty jumps, drift aerials, pushback on hit.

## 5–8. Combat & aura
Gale push on side attack; tornado special; wind ribbons at aura_core.

## 9. VFX sockets
Sash, wing_l, wing_r, hands for gust bursts.

## 13. Required assets
`kaia-windrow.blend`, `kaia-windrow.glb`

## 14. Acceptance
Wind sash visible at default zoom; aerial attacks show leg/arm extension.
