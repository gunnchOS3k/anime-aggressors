# Rook Ironside — Production Spec

## 1. Fantasy
Heavy armor brawler — space control with shoulder checks and ground pounds.

## 2. Silhouette
Broad shoulders, heavy boots, helm read, earth-brown/indigo palette.

## 3. Body type
Large / broad — slowest roster weight class feel.

## 4. Movement identity
Plodding but armored — wins trades, loses runaways.

## 5. Difficulty
- Beginner: high damage, simple inputs
- Intermediate: shield and spacing
- Advanced: ledge guard with down attack

## 6. Attack style
Shoulder ram side attack, overhead smash, low boot sweep.

## 7. Special style
Seismic slam — armor super with screen shake.

## 8. Aura visual language
Dust and stone chips orbit shoulders.

## 9. VFX sockets
- aura_core: chest plate
- right_hand: gauntlet
- weapon_tip: boot (down attack)

Required: `rook-ironside.blend`, `rook-ironside.glb`

Acceptance: shoulder silhouette distinct from Ember/Juno at camera distance.
