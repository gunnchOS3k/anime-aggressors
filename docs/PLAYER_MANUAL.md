# Player Manual — Anime Aggressors

**Last updated:** 2026-07-01

Anime Aggressors is a local platform fighter. Two players (or one player vs CPU) fight on a stage, deal damage, launch opponents, and score KOs until stocks run out or time expires.

---

## Starting a match

### Start Game (full setup)

1. **Home** → **Start Game**
2. **Fighter Select** — pick P1 and P2 (all 7 fighters available)
3. **Stage Select** — pick a production stage
4. **Ready Check** — confirm loadouts, optional CPU for P2, test inputs
5. **Battle**

### Quick Play

**Home** → **Quick Play** — launches a stock match immediately with default fighters and Skyline Arena.

### Training

**Home** → **Training Mode** — practice on Training Grid with a dummy opponent.

---

## Match rules

- **Stock:** Lose all stocks → eliminated. Last fighter standing wins.
- **Damage %** increases knockback — higher % means longer launches.
- **Blast zones** outside the stage count as KOs.
- **Respawn** grants brief invulnerability.

---

## Pause menu

- Resume
- Rematch
- Change fighters / stage
- Return home

---

## Reset Game State

On **Home**, use **Reset Game State** if menus or battle launch behave incorrectly (clears bad saved setup).
