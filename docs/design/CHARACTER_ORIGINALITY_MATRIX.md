# Character Originality Matrix

Original fighter identities for Anime Aggressors. Inspiration categories inform combat grammar only — no franchise costumes, move names, or signature sequences are reproduced.

## Ember Vale

| Dimension | Definition |
|-----------|------------|
| Visual theme | Volcanic runner with ember-lined cloak plates and heat-scarred boots |
| Silhouette | Forward-leaning sprinter, wide shoulders, flame-tail hair volume (angular, not licensed) |
| Power lane | Fire / heat / combustion rushdown |
| Personality | Impulsive, competitive, laughs through danger |
| Movement identity | Fast ground bursts, short hops, wall-rush angles |
| Neutral game | Layered jabs into dash-in grabs; trades space for pressure |
| Advantage state | Burn stacks, flame trails, bigger explosive finishers at high aura |
| Disadvantage weaknesses | Overextends on whiff; burn setups need setup time |
| Aura behavior | Heat shimmer, rising ember particles, intensifying trail length |
| Projectile behavior | Short-lived fire blobs with arcing lobs at high aura |
| Grab / throw | Forward rush grab; throws launch at sharp diagonals |
| Recovery | Fast fall with burn trail but vulnerable landing lag |
| Counterplay | Shield spacing, anti-air at jump apex, punish dash endlag |
| Inspiration categories | Shōnen rushdown, fire archetypes, explosive finishers |
| Explicit differences | No licensed spiky blond hero, no named fire techniques, original costume geometry |
